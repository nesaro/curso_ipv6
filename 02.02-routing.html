Las distintas opciones de enrutado

  • Estático
  • RIP (IGP)
  • IS-IS
  • OSPF (IGP)
  • Multiprotocol BGP (EGP)

RIPng

  • Pensado para redes medianas
  • Utiliza el puerto 521 UDP
  • Creación de la tabla con un algoritmo simple

OSPF

  • Pensado para superar las limitaciones de RIP
  • Utiliza el algoritmo de Dijkstra
  • Converge rápidamente
  • Fue necesario crear una nueva revisión para IPV6

IS-IS

  • Muy similar a OSPF
  • Utiliza el algoritmo de Dijkstra
  • Es menos hablador que OSPF, más usado en ISP
  • No utiliza IP, esta en la capa de red de OSI. No ha sido modificado para IPv6

BGP

  • Conecta sistemas autónomos
  • Usa TCP 179
  • Consiste en la comunicación entre los distintos AS de las rutas conocidas
  • No hay diferencias sustanciales en la implementación de IPv6
