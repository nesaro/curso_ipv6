Cabecera de Destino y Hop By Hop

  • Cabecera de Destino
    • Código de próxima cabecera: 60
    • Para Ipv6 Movil y algunas aplicaciones
  • Cabecera Hop by hop
    • Código de próxima cabecera: 0
    • Se utiliza para establecer los Jumbo Datagram
    • Se utiliza para el protocolo Multicast listener discovery
    • Siempre debe ser la primera extensión
